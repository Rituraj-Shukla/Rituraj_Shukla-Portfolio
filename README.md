## 🚀 Getting Started

### **Prerequisites**
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Code editor (VS Code, Sublime Text, or similar)
- Node.js & npm (optional, for SCSS compilation)

### **Quick Setup**

#### **1️⃣ Clone the Repository**
```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

#### **2️⃣ Open Locally**
```bash
# Navigate to the Rituraj folder
cd Rituraj

# Open index.html in your browser
# Option 1: Direct open
open index.html

# Option 2: Using Python server (recommended)
python -m http.server 8000
# Then visit http://localhost:8000
```

#### **3️⃣ SCSS Development (Optional)**
```bash
# Watch and compile SCSS files
sass assets/scss/app.scss assets/css/styles.css --watch

# One-time compilation
sass assets/scss/app.scss assets/css/styles.css
```

## 🔧 Customization Guide

### **1. Update Personal Information**
Edit `index.html` to replace:
- Name and title
- Professional summary
- Contact information
- Social media links

### **2. Modify Color Scheme**
Edit `assets/scss/abstracts/_variable.scss`:
```scss
// Primary colors
$primary-color: #your-color;
$secondary-color: #your-color;

// Neutral palette
$text-dark: #1a1a1a;
$text-light: #ffffff;
```

### **3. Update Portfolio Projects**
Replace image paths in portfolio grid section with your project screenshots and update project links.

### **4. Customize Services**
Modify the services section to reflect your specific expertise areas.

### **5. Add Your Analytics**
Integrate Google Analytics or your tracking solution in `<head>` section.
